Graphs

Bar Graphs

Bar graphs are visual representations of data using rectangular bars. The length or height of each bar corresponds to the frequency or magnitude of the data it represents.

VIEW

Count Plots

A count plot displays the frequency of categorical data. It shows the number of occurrences of each category in a dataset using bars, where the height of each bar represents the count.

VIEW

Pie Charts

A pie chart is a circular graphical representation of data, where each slice represents a proportion or percentage of the whole. It is commonly used to show overall composition of a dataset.

VIEW

Miscellaneous

Scatterplots use points to represent the data, while heat maps use color to represent the density of data points in different regions. Both can be used to identify patterns and trends in the data.
